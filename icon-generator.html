Icon Generator for Finca Termópilas

Este archivo te ayudará a generar los iconos necesarios para el sitio web de Finca Termópilas.

Paso 1: Favicon (32x32)

Necesitas crear un archivo favicon.png de 32x32 píxeles.

  1. Ve a Favicon.io
  2. Sube la imagen hero-bg.jpg o cualquier otra imagen que quieras usar como favicon
  3. Descarga el paquete de favicon
  4. Extrae el archivo favicon.png y colócalo en la carpeta assets/images/

Paso 2: Apple Touch Icon (180x180)

Necesitas crear un archivo apple-touch-icon.png de 180x180 píxeles.

  1. Ve a Resize Image
  2. Sube la imagen hero-bg.jpg o cualquier otra imagen
  3. Redimensiona a 180x180 píxeles
  4. Descarga la imagen y renómbrala como apple-touch-icon.png
  5. Coloca el archivo en la carpeta assets/images/

Paso 3: Iconos PWA (192x192 y 512x512)

Necesitas crear dos archivos: icon-192x192.png y icon-512x512.png.

  1. Ve a Resize Image
  2. Sube la imagen hero-bg.jpg o cualquier otra imagen
  3. Redimensiona a 192x192 píxeles y descarga como icon-192x192.png
  4. Repite el proceso para crear una imagen de 512x512 píxeles y descarga como icon-512x512.png
  5. Coloca ambos archivos en la carpeta assets/images/

Paso 4: Capturas de pantalla para PWA (1280x720)

Necesitas crear dos archivos: screenshot1.jpg y screenshot2.jpg.

  1. Ve a Resize Image
  2. Sube la imagen hero-bg.jpg y redimensiona a 1280x720 píxeles
  3. Descarga como screenshot1.jpg
  4. Repite el proceso con la imagen couples.jpg y descarga como screenshot2.jpg
  5. Coloca ambos archivos en la carpeta assets/images/

Nota: También puedes usar cualquier editor de imágenes como Photoshop, GIMP, o incluso herramientas en línea como Canva para crear estos iconos.

Verificación

Una vez que hayas creado todos los iconos, deberías tener los siguientes archivos en la carpeta assets/images/:

  • favicon.png (32x32)
  • apple-touch-icon.png (180x180)
  • icon-192x192.png (192x192)
  • icon-512x512.png (512x512)
  • screenshot1.jpg (1280x720)
  • screenshot2.jpg (1280x720)
